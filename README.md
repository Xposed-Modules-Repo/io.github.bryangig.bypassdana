# Bypass DANA
A Module to Bypass DANA's Integrity Check.

### Tested
Support for DANA Stable

### Installation
1. Install Xposed (Alternative: EdXposed, LSPosed)
2. Install the release `.apk`
3. Enable the module and force stop DANA.
4. Enjoy!
